오라클 26ai : 최신 장기 버전(LTR). 장기 버전은 몇 년에 한 번 나오기 때문에 실무에서는 이전LTR에서 다음LTR로 시스템 업그레이드함.
java17 사용하는 이유 : 가장 폭넓은 외부 라이브러리 호환성을 가진 java버전이라서.
웹 주소가 소문자인 이유 : 일부 리눅스 시스템에서는 대문자와 소문자를 구별하고, 사용자가 헷갈릴 수 있음\

객체 관리(IoC): 스프링이 알아서 객체를 만들고 없앰.
의존성 주입(DI): '만들어진' 객체들을 서로 연결(사용할 수 있도록 서로 알도록)해 줌.(@Autowired)
@controller, @service등등 : 그 클래스의 메소드를 싱글톤으로 만든다(객체 생성)
import : 클래스의 위치를 알려줌.

static : 정적 멤버는 인스턴스 생성 하지 않음
싱글톤 : 인스턴스 딱 하나 생성함

필드명으로 created_at 이렇게 있으면 년/월/일만 필요한지, 시분초또한 필요한지 모름

string pool은 직접 작성한 문자열("hello")에만 해당됨
보안 때문에 string이 걱정된다면 메모리에 잔류하지 않도록 char[]사용
